-

[

## 📙 2. General Form

A recurrence relation expresses (a_n) as a function of previous terms:
[

a_n =3D f(a_{n-1}, a_{n-2}, ..., a_{n-k}) + g(n)

- 
(f(\cdot)) -> function of previous terms

- 
(g(n)) -> function of n (optional)

-

| Type | Description | Example |
